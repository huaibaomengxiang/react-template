# react-template
这是一个react项目的模板仓库,技术栈包括react、react-router4、redux、webpack4  
##项目目录
```
.
├── build ---------------------- webpack配置文件
├── src ------------------------- 源码目录
    ├── component --------------- 公共组件
    ├── images ------------------ 图片文件
    ├── store ------------------- redux
        ├── actions ------------- actions
        ├── reducer ------------- reducer
    ├── view -------------------- 视图组件
    ├── index.js ---------------- 入口文件
    ├── index.less -------------- 全局样式
    ├── router.js --------------- 路由配置
├── babelrc ----------------------------- babel配置
├── .gitignore ---------------------------- git忽略文件配置
└── package.json ------------------------------- 都懂的
├── postcss.config.js -------------------------- postcss配置
├── webpack.config.js -------------------------- 判断环境
```
